Ingeniería en sistemas computacionales

Universidad Alvart 2019 - 2023

Soy graduado de la carrera de Ingeniería en Sistemas Computacionales de la Universidad Alvart, donde mi travesía educativa fue un viaje de descubrimiento y maestría en diversas áreas de la informática. A lo largo de mi formación, construí una sólida base en programación, comenzando desde cero con pseudocódigo en PSEINT. Progresé hacia lenguajes de programación fundamentales como C++ y Java, abordando desafíos de algoritmos para perfeccionar mi lógica de programación.


El mundo del desarrollo de software se abrió ante mí con C# en Visual Studio, donde adquirí habilidades en programación orientada a objetos, algoritmos de ordenamiento y diseño de interfaces utilizando las herramientas proporcionadas por este entorno de desarrollo integrado.


Mi inmersión en el desarrollo de software continuó con el dominio de bases de datos relacionales, empleando gestores como MySQL con herramientas como Workbench y PHPMyAdmin. No solo aprendí a realizar operaciones CRUD con distintos proyectos, sino también a implementar medidas de seguridad cruciales, como la prevención de SQL Injection y la encriptación de contraseñas.


Exploré más allá de las bases de datos relacionales, adentrándome en el mundo de las bases de datos no relacionales, comprensión de documentos y el manejo del formato JSON.


La fase siguiente de mi educación me llevó al universo del desarrollo web, comenzando desde lo más básico con HTML, CSS y JavaScript. Rápidamente avancé para conectar con Firebase, y amplié mi conocimiento al mundo del desarrollo web con Python a través de Django. Además, fui introducido a PHP, donde no solo dominé la creación de operaciones CRUD y cifrado de datos, sino que también me familiaricé con las particularidades de mysqli y PDO, y se me enseñó el framework Laravel.


Continuando con mi diversificación en el ámbito del desarrollo, exploré el desarrollo de aplicaciones móviles utilizando Kotlin y Java en el entorno Android Studio. La experiencia previa en desarrollo web y Firebase resultó invaluable, proporcionándome un contexto sólido y habilidades transferibles para afrontar este emocionante paso.


Mi viaje educativo ha sido un proceso continuo de crecimiento, diversificación y adaptación a nuevas tecnologías, lo cual me ha dotado de una perspectiva integral y habilidades versátiles en el vasto campo del desarrollo de software.

Desarrollador fullstack

Sistema estatal DIF gobierno de Puebla 2022 - 2023

Durante mi servicio social, lideré el desarrollo de una aplicación web destinada a la gestión eficiente de reportes dentro del área de soporte técnico del sistema estatal DIF en diversas sedes.


Descripción del Proyecto:

La aplicación que diseñé y desarrollé permite la generación, gestión y eliminación de reportes provenientes de distintas áreas y sedes de la institución. Esta plataforma cuenta con roles diferenciados para usuarios, administradores y técnicos, cada uno con funciones específicas.


  • Usuario: Capaz de generar reportes, con la capacidad de incluir imágenes en sus registros.
  • Administrador: Con la facultad de asignar técnicos a cada reporte, así como eliminar y crear nuevos reportes.
  • Técnico: Encargado de visualizar los reportes asignados por el administrador, indicar el estado del servicio y marcarlos como completados.

La creación de un reporte incluye detalles cruciales como la fecha y hora de levantamiento, un token único para identificación, y registra el nombre y área del usuario. Al marcar un reporte como completado, se permite seleccionar la fecha y hora de finalización del servicio.


Este proyecto abarcó:

  • Diseño y conexión a la base de datos.
  • Implementación de una base de datos relacional.
  • Desarrollo de un diseño responsivo.
  • Gestión de usuarios y roles.
  • Ingeniería de software.
  • Despliegue efectivo de la aplicación.
